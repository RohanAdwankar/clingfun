# clingfun is an infinite generator of fun comp ling problems

i've been playing a lot of chess puzzles recently and thery are great because they are bite sized but also you have to think about it a bit

this is a simple site which runs rust in wasm, statically hosted on github pages, which generates infinite amounts of these problems

the general structure is every problem it presents you with a cfg or fsa and you have to answer whether the given string matches that language

the game works like chess puzzles in that the faster you answer the more points you get
